# ASCII Art Plugin

将图片转换为ASCII艺术的插件。

## 功能

- 当回复中包含指定标签且带有图片时，自动将图片转换为ASCII艺术
- 支持各种图片格式（JPEG、PNG、GIF等）
- 可配置ASCII艺术的宽度、高度、字符集和触发标签

## 配置

```yaml
enabled: true           # 是否启用插件
priority: 40            # 插件优先级
width: 60              # 固定宽度（当auto_scale为false时使用）
height: 30             # 固定高度（当auto_scale为false时使用）
chars: " ░▒▓█"         # 用于生成ASCII艺术的字符集，使用Unicode块字符获得更细腻效果
tag: "#ascii"          # 触发ASCII转换的标签，可自定义
auto_scale: true       # 自动根据图片比例调整尺寸
max_width: 80          # 最大宽度限制
max_height: 40         # 最大高度限制
preserve_aspect: true   # 保持宽高比
```

## 自动比例功能

插件支持两种尺寸模式：

### 1. 自动缩放模式（推荐）
- `auto_scale: true` - 根据图片特点自动选择最优尺寸
- **竖图优化**：高宽比 > 1.5 时，优先保证高度显示完整
- **横图优化**：高宽比 < 0.6 时，优先保证宽度显示完整  
- **方图平衡**：高宽比在 0.6-1.5 之间时，平衡宽高显示
- 智能适应不同图片类型，获得最佳显示效果

### 2. 固定尺寸模式
- `auto_scale: false` - 使用配置的固定宽度和高度
- `preserve_aspect: true` - 在固定尺寸内保持图片宽高比
- `preserve_aspect: false` - 强制拉伸到指定尺寸

## 字符集说明

插件支持多种字符集风格：

1. **Unicode块字符**（推荐）：`" ░▒▓█"` - 提供最细腻的渐变效果
2. **传统ASCII字符**：`" .:-=+*#%@"` - 经典ASCII艺术风格
3. **简化字符**：`" .-+#"` - 简单清晰的效果
4. **自定义字符**：你可以定义任何字符组合

## 使用方法

1. 在回复中添加图片
2. 在回复文本中包含配置的触发标签（默认为 `#ascii`）
3. 机器人会自动分析图片比例并选择最优尺寸进行转换
4. 获得保持原图比例且显示效果最佳的ASCII艺术

## 最佳实践

- **人像照片**：自动识别为竖图，优化纵向显示
- **风景照片**：自动识别为横图，优化横向显示  
- **正方形图**：平衡处理，获得最佳整体效果
- **极端比例**：自动调整以确保关键内容完整显示

## 自定义触发标签

你可以在 `config.yaml` 中修改 `tag` 字段来自定义触发标签：

```yaml
tag: "#art"      # 使用 #art 作为触发标签
tag: "转ascii"    # 使用中文标签
tag: "ascii me"  # 使用多词标签
```

## 依赖

- Pillow: 用于图片处理
- aiohttp: 用于下载图片

## 技术特性

- **图像增强**：自动锐化、对比度和亮度调整
- **直方图均衡化**：提高图像细节表现
- **高质量重采样**：使用LANCZOS算法保持图像质量
- **智能比例调整**：针对不同字符集优化宽高比
- **Unicode支持**：支持块字符获得更细腻的渐变效果

## 注意事项

- 图片会被下载到内存中处理，对于大图片可能消耗较多内存
- 使用Unicode块字符时效果最佳，但需要终端/显示器支持
- ASCII艺术的质量取决于原图的对比度和内容
- 建议原图具有清晰的轮廓和较好的对比度以获得更好的效果
- 文件大小限制为10MB
