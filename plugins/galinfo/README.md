# galinfo 插件

本插件为 misskey-ai 平台提供 Galgame 信息查询能力，基于月幕Gal API。

## 指令
- `/gal <游戏名>` 查询指定 Galgame 信息
- `/galvague <游戏名>` 模糊查询 Galgame 信息

## 配置
请参考 `config.yaml.example` 进行配置。

## 依赖
- aiohttp
- aiofiles
- pillow
